## 🚀 Usage

* ### Install dependencies
```sh
npm install
```

* ### Start developement Server
```sh
npm run dev
```


## 📖 Features

- [x] Create new Tasks 
- [x] Edit Task 
- [x] Delete Task 
- [x] Mark tasks as completed
- [x] Dark Mode
- [x] LocalStorage 


## 💻 Tech Stack

- HTML
- CSS
- JS
- React.js (Vite)

Used the following features to make the app responsive and pass data between
components.

- Context Api
- useState
